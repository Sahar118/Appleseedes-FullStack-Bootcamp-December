ההבדל בין ID לבין CLASS

לכל DIV צריך להיות ID אחד ואי אפשר להשתמש באותו ID ביותר מפעם אחת, לעומת זאת ה CLASS אפשר להשתמש באותו אחד ביותר ממקום אחד הוא עוזר לנו בדרך כלל לעיצובים,
כאשר אנו רוצים לעצב כמה דברים את אותו הדבר, ניתן להם את אותו CLASS 


ההבדל ביען DIV ל SPAN 
SPAN משתמשים בו בעיקר כאשר אנו רוצים להוסיף CLASS לתוף פסקה, ואז הוא לא יורד שורה אוטומטי כמו ב DIV
